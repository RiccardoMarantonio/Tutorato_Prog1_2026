# Struct/Puntatori/Map - Hard - Miglior Studente

Su un registro mappa->puntatore con storico voti, individua lo studente con media valida piu alta.

Regole:
- considerare solo voti nel range 0..30
- ignorare record nil o senza voti validi
- in caso di parita, scegliere il nome lessicograficamente minore
- se nessun candidato valido, restituire nome vuoto e media 0
