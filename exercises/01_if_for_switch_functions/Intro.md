# Numeri Pari

Scrivi un programma che prende un numero `n` std input e stampa, uno per riga, tutti i numeri pari da `0` a `n`.
Implementa una funzione isEven che prende come input un intero positivo e restituisce `true` se è pari, `false` se è negativo

## Esempio

Input:
```
10
```

Output:
```
0
2
4
6
8
```
