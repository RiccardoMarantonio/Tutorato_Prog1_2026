# Tabelline

Scrivi un programma che stampa le tabelline da 1 a `n` (letto da stdin).

## Formato Output
Ogni tabellina deve essere stampata su una riga singola nel formato:
```
1 x 1 = 1
1 x 2 = 2
...
1 x 10 = 10

2 x 1 = 2
2 x 2 = 4
...
```

Dopo ogni tabellina, stampa una riga vuota.

## Esempio

Input:
```
3
```

Output:
```
1 x 1 = 1
1 x 2 = 2
1 x 3 = 3
...
1 x 10 = 10

2 x 1 = 2
2 x 2 = 4
...
2 x 10 = 20

3 x 1 = 3
3 x 2 = 6
...
3 x 10 = 30
```

## Vincoli
- `1 <= n <= 12`
- Ogni tabellina arriva fino a 10
- Scrivi tutto il programma da zero
- Usa cicli `for` annidati
